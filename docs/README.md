# Documents

このディレクトリには、`Flyby Control Challenge` の設計・仕様・当日運用資料を置きます。

## Files

- `DECISIONS.md`：設計判断の一覧。実装方針に迷った場合の根拠。
- `SPEC.md`：v0.1 実装仕様。Codex / coding agent に最初に読ませる仕様書。
- `TECHNICAL_MODEL.md`：数値・物理モデルと点数計算を、実装に対応する数式でまとめた技術資料。
- `event/WORKSHEET.md`：団員向けA4 1枚ワークシート案。
- `event/MENTOR_CHEATSHEET.md`：メンター向けA4 1〜2枚チートシート案。
