# Flyby Control Challenge

宇宙少年団の例会「はやぶさ2『トリフネフライバイ』について学ぼう」で使用する、教育用Webアプリの企画・仕様リポジトリです。

本アプリは、はやぶさ2・小惑星トリフネフライバイを題材に、団員が「運用チーム」になったつもりで、`ΔV` と `Bプレーン` を使ってフライバイ通過位置を調整する体験を行うための非公式教育シミュレーションです。

## 非公式教材としての注意

本アプリは、はやぶさ2・トリフネフライバイを題材にした非公式の教育用Webアプリです。JAXA/ISASおよびはやぶさ2プロジェクトの公式・公認・監修コンテンツではありません。

ゲーム内の軌道、`ΔV` 操作、Bプレーン表示、スコア、危険判定は、フライバイ探査の考え方を学ぶために単純化したものです。実際のはやぶさ2運用、実軌道、姿勢制御、観測機器挙動、観測シーケンスを再現するものではありません。

## 学習目標

団員に覚えて帰ってほしい中心語は次の2つです。

- `ΔV`：探査機の速度を少し変えること。
- `Bプレーン`：小惑星の近くをどのあたりで通るかを見る地図。

体験として理解してほしいことは次の通りです。

- 小さな `ΔV` でも、時間が経つと通過位置が変わる。
- 近いほど観測しやすいが、近づきすぎると危険である。
- よい作戦は、観測と安全のバランスを考えて決める。
- 本物の探査では、さらに姿勢制御、カメラ指向、画像フィードバック、通信・電力方向なども関係する。

## v0.1 の設計方針

v0.1では、当日確実に動くことを最優先にします。

- 結果カードを見れば、団員・メンターが作戦を説明できるようにする。

## v0.2 試作中の発展ルール

現在のアプリには、活動中の作戦検討を深めるため、次の発展ルールも実装しています。

- `T-24 h` と `T-6 h` の2回に分けてΔV運用を行う。
- 2回で使えるΔVの合計は `0.30 m/s`。
- 第1回修正後の広い予測分布と、観測更新後の狭い最終予測分布を比較する。
- 追加観測で判明する中心位置は、第1回修正後の予測分布からチャレンジごとに変化する。
- 最終中心点だけでなく、危険領域とおすすめ観測領域へ入る確率を評価する。
- `第1回運用 → 第2回運用 → 運用結果` の3画面を順に進める。
- 感度行列を使い、ΔVの大きさと方向に応じてBプレーン誤差楕円の大きさ・向きを変える。
- 第2回運用では、修正なしの場合と現在の入力による効果を比較表示する。
- 各運用で速度変更前後の通り道の予想範囲を表示し、その広さの変化を確認できる。
- Bプレーンの主要距離圏表示と線形ズーム切替を備える。

この予想範囲と確率評価も、実運用を再現するものではなく、限られたΔVと噴射のずれを考えるための教育用簡略モデルです。

## ローカルで開く

`index.html` を直接ダブルクリックすると、ブラウザの制限によりアプリを読み込めません。

Windowsでは、リポジトリ直下の `START_APP.cmd` をダブルクリックしてください。初回のみ必要なパッケージをインストールし、ローカルサーバーを起動してブラウザを開きます。起動中はコマンド画面を閉じないでください。

ターミナルから起動する場合：

```powershell
npm.cmd install
npm.cmd run start
```

## 基本思想

既存可視化は「見る教材」、本アプリは「考える・操作する教材」として役割分担します。

本アプリは、実ミッション再現ソフトではありません。実際の探査で大事な考え方の一部、特に `ΔV` と `Bプレーン` による通過位置調整を、団員が自分の手で試せるようにする教材です。

公開情報を参考に、トリフネ本体の等価半径約225 m・最大半径目安約420 m、フライバイ中心距離1〜10 kmを距離表示へ反映しています。ゲーム内では、安全のための赤い危険エリアを750 m、近くで見られるエリアを1〜3 km、安全に見られるエリアを3〜10 kmとして簡略評価します。

噴射のずれは、固定のずれ・ΔV量に比例するずれ・大きな噴射への追加のずれから計算します。第2回修正は第1回修正より細かく操作でき、一度に大きく動かすか、ΔVを残して最後に仕上げるかが作戦になります。

第2回運用画面では、第1回ΔV後の通り道の予想範囲が、追加観察によってせまくなる様子を表示します。第2回ΔVは操作範囲を各軸 `±0.10 m/s`、最小目盛を `0.005 m/s` とし、最後の安全を整える細かな修正として操作できます。

通り道の予想範囲、安全の判定、面積はすべて3σで統一しています。マップでは中心ほど濃く、3σの外側へ向かって薄くなる分布として表示し、安全判定に使う3σ外縁を薄い破線で示します。

公開資料に示されたフライバイ時のトリフネ位置不確かさ `14.6 km` は、Bプレーン上の長半径・短半径として直接使用していません。教育用の縮尺として、第1回修正前は3σで長い方向 `7.0 km`・短い方向 `3.5 km`、追加観測後は長い方向 `2.0 km`・短い方向 `0.8 km` としています。楕円は方向差を持ち、各ΔVの大きさと方向によってさらに広がり、回転します。

結果スコアは、最終分布だけを評価します。最終分布から160本の通り道を取り出し、それぞれをトリフネ中心からの距離に応じて採点した平均値を使います。操作の途中経過、2回目で改善した量、残したΔV量は得点へ加えません。

- 科学的意義得点：50点
- 安全得点：50点

科学的意義得点は中心距離1 km付近を頂点として近いほど高く、安全得点は危険円から3 kmまでS字状に増加して遠いほど高くなります。2つの得点は反対方向へ動くため、100点を取れる作戦はありません。スコアは唯一の正解ではなく、この教材で設定した評価条件による比較の目安です。最終的な3σ予想楕円が赤い危険エリアへ重なる場合、合計点は最大59点になります。
